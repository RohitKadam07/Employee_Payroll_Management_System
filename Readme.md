# Employee Payroll Management System  
Designed and implemented a payroll management system that allows creating employees, adding them to the payroll, calculating their salaries based on employee type, and managing their removal.
Applied core Object-Oriented Programming concepts including Abstraction, Inheritance, Polymorphism, and Encapsulation to ensure modularity and scalability.
Enhanced system usability by providing clear workflows for salary calculation and employee management.

## Features:
Abstract Employee class serving as the foundation for different employee types.
Concrete subclasses FullTimeEmployee and PartTimeEmployee with salary calculation.
Demonstrates core OOP concepts with practical coding examples.
